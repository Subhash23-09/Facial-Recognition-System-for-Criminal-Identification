System Description

This AI-powered system uses cutting-edge face recognition technology to help identify potential criminals and locate missing individuals. It leverages:

  • MTCNN for face detection
  • Bicubic Interpolation & Bilateral Filtering for face enhancement
  • ViT (Vision Transformer) for face embeddings
  • Cosine Similarity for accurate face matching
